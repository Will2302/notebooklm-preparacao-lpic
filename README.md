# notebooklm-preparacao-lpic
NotebookLM - certificações LPIC‑1 e LPIC‑2

Contexto e Objetivos
Este repositório documenta o uso do NotebookLM como ferramenta de aprendizagem ativa para a preparação das certificações LPIC‑1 e LPIC‑2 (LPI).
O objetivo é:

Transformar materiais de estudo (PDFs, resumos, links oficiais) em um caderno temático estruturado no NotebookLM.

Organizar resumos, glossário e prompts reutilizáveis para revisão contínua.

Deixar claro o raciocínio por trás dos prompts e como foi possível ajustar interações com a IA para obter respostas mais acertadas.
